## 🔧 Bug FIX NGAY trong phiên (commit kèm)

1. **P0 — Fake payment fail ngầm, sinh 3 đơn zombie**: UI tính total
   934.400đ nhưng backend tạo đơn 925.056đ (server tự áp discount 1%
   trên race 257). Mobile gửi amount client → "Amount must be 925.056"
   → toast biến mất, user kẹt. Fix: SDK đọc `total_price` từ response
   create (field thật — `total`/`total_amount` không tồn tại) + checkout
   dùng server total. Zombie orders đã dọn (2 cancel, 1 thanh toán).
2. **P0 — NativeSignFlow crash "html.replace is not a function"**:
   `/pub/race-skip-all-liability-html` trả double-nested **S3 URL**
   (`{data:{data:"https://...html"}}`), không phải HTML inline. Fix:
   unwrap + fetch S3 → template render + ký thành công.
3. **P0 — DateField sheet vỡ trong form**: project BottomSheet =
   @gorhom non-modal, chỉ sống ở root màn → nằm trong ScrollView là sập
   thành dải. Fix: DateField chuyển sang RN `Modal` thuần — picker 3 cột
   hoạt động đẹp.
4. **P1 — "View ticket" sau thanh toán dead-end**: `order.ticketId`
   không bao giờ tồn tại, fallback orderId → trang vé lỗi. Fix: resolve
   vé thật qua listMyTickets match orderId, fallback về list /tickets.
5. **P2 — Badge "—" cho vé CHECKEDIN**: backend fetch-by-user trả
   `CHECKEDIN` (không gạch dưới) — normalise chỉ fold `CHECK_IN`. Fixed.
6. **P2 — "##" kép trên màn success**: order_number đã có "#", screen
   prepend thêm. Fixed (strip trước khi prefix).
